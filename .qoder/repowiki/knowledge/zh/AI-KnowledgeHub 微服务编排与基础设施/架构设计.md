- **依赖治理**：根 `pom.xml` 统一管理 Spring Boot/Cloud 版本及各子模块依赖，确保技术栈一致性。
- **流量入口**：`gateway-service` 作为唯一外部入口，集中处理 JWT 鉴权、IP 限流及路由转发，隔离后端微服务。
- **异步通信**：基于 RabbitMQ Fanout Exchange 实现 `article-service` 与 `ai-service` 的解耦，支撑文章发布后的标签提取与合规检测。
- **环境编排**：`docker-compose.yml` 一键拉起 MySQL、Redis、RabbitMQ 共享中间件，并通过健康检查保障服务启动顺序。
- **公共契约**：`common` 模块以 Spring Boot Starter 形式提供 JWT 工具、MyBatis-Plus 增强及统一响应封装，被所有业务服务依赖。
