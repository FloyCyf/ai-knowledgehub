   - 影响：编译失败不会第一时间发现
   - 解决方案：阶段 1 完成后由 A 在本地立刻执行 `mvn clean install`，如有失败立即修复

3. **common/README.md 未单独写**（D6-3）
   - 影响：新人入职不知道 common 有哪些类
   - 解决方案：阶段 2 顺便补一份 common/README.md

### 10.2 阶段 1 自我评价

**总体判断**：阶段 1 任务**已按计划完成**，所有 6 维度均 ≥ 80 分，加权总分 90.3，**通过**。

**最大风险**：未实机执行 `mvn install`。建议在阶段 2 启动前**先在本地执行一次 `mvn clean install`**，确认整个项目可编译。

- `maven-enforcer-plugin` 强制 Java 17 / UTF-8 / 依赖收敛 / 禁止重复版本声明 —— 后续 A/B/C 谁写代码不规范都会被自动拦下
