## 九、阶段 2 关键代码片段

### 9.1 注册（BCrypt + 用户名唯一性）

```java
@Transactional(rollbackFor = Exception.class)
public Long register(RegisterDTO dto) {
    String username = dto.getUsername().trim();
    Long count = userMapper.selectCount(new LambdaQueryWrapper<User>()
            .eq(User::getUsername, username));
    if (count != null && count > 0) {
        throw new BusinessException(ResultCode.USERNAME_EXISTS);
    }
    User user = new User();
    user.setUsername(username);
    user.setPasswordHash(passwordEncoder.encode(dto.getPassword()));  // BCrypt
    user.setRole(HeaderConstants.ROLE_USER);
    user.setStatus("ENABLED");
    userMapper.insert(user);
    return user.getId();
}
```

### 9.2 登录（防用户名枚举 + JWT 签发）

```java
public LoginVO login(LoginDTO dto) {
    String username = dto.getUsername().trim();
    User user = userMapper.selectByUsername(username);
    if (user == null || !user.isEnabled()) {
        throw new BusinessException(ResultCode.LOGIN_ERROR);  // 统一错误，防止枚举
    }
    if (!passwordEncoder.matches(dto.getPassword(), user.getPasswordHash())) {
        throw new BusinessException(ResultCode.LOGIN_ERROR);
    }
    String token = JwtUtil.generateToken(user.getId(), user.getUsername(), user.getRole(), jwtProperties);
    return LoginVO.builder()
            .token(token)
            .expiration(System.currentTimeMillis() + jwtProperties.getExpirationMillis())
            .user(toVO(user))
            .build();
}
```

### 9.3 注销（Token 黑名单）

```java
public void logout(String token) {
    if (token == null || token.isBlank()) return;
    try {
        Claims claims = JwtUtil.parseToken(token, jwtProperties);
        tokenBlacklistService.blacklist(claims);
    } catch (AuthException e) {
        // Token 已过期无需加入黑名单
    }
}
```

### 9.4 个人信息（SecurityUtils 替代 @RequestHeader）

```java
@GetMapping("/profile")
public ApiResponse<UserVO> profile() {
    Long userId = SecurityUtils.requireCurrentUserId();  // 自动从 X-User-Id 头取
    return ApiResponse.success(userService.getProfile(userId));
}
```
